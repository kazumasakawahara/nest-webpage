01

技術構成

フロントエンド / 基盤
  • Astro v6 — 静的書き出し＋Cloudflare adapter（会員エリアのみ SSR）
  • Vanilla CSS — デザイントークン（緑・クリーム・テラコッタ／明朝＋ゴシック）
  • Markdown Content Collection — News をスタッフが追記可能
  • Vitest 16件／Playwright E2E 基盤
会員エリア / バックエンド
  • Supabase（東京リージョン）— DB＋Storage（機関誌・議事録は Private）
  • 自前マジックリンク認証 — Web Crypto 採用で Cloudflare Workers 互換
  • Resend — 認証メール送信（現在は開発用ドメイン）
  • RLS deny-by-default＋ロールベースアクセス制御（RBAC）
