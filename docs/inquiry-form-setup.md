# お問い合わせフォーム（Googleフォーム＋QR）セットアップ手順

お問い合わせページ（`/contact/`）の埋め込みフォームと QR コードを本番表示にするまでの手順です。

## 全体像

```
[Apps Script を実行] → Googleフォームが自動生成され「公開URL」が発行される
        ↓ 公開URLをコピー
[contact.astro の FORM_VIEW_URL に貼り付け] → デプロイ
        ↓
ページに「埋め込みフォーム」と「QRコード」が自動表示される
（QRはビルド時に公開URLから自動生成。第三者サービス不使用・インラインSVG）
```

## 手順

### 1. Googleフォームを作成する

1. <https://script.google.com/> を開き「新しいプロジェクト」を作成
2. `scripts/create-nest-inquiry-form.gs` の中身を全部コピーして貼り付け（既存コードは消す）
3. `NOTIFY_EMAIL`（回答通知の送信先）を確認
   - **現在はダミー**（`inquiry@example.nponest.org`）。本番アドレスが決まったら書き換える
4. 関数 `createNestInquiryForm` を選んで ▶ 実行 → 初回は権限承認
5. 実行ログ（`Ctrl + Enter`）に出る **公開URL** と **編集URL** を控える

作成されるフォームの項目（設計済み）:

| # | 項目 | 種別 | 必須 |
|---|------|------|------|
| 1 | お問い合わせ種別（B型見学・GH空き状況・入会/寄付/ボランティア・取材/講演・当事者活動・親なき後・その他） | 単一選択 | ✅ |
| 2 | お名前 | 短文 | ✅ |
| 3 | ふりがな | 短文 | |
| 4 | ご所属・団体名 | 短文 | |
| 5 | メールアドレス（形式チェック付き） | 短文 | ✅ |
| 6 | 電話番号 | 短文 | |
| 7 | ご連絡方法のご希望（メール/電話/どちらでも） | 単一選択 | |
| 8 | お問い合わせ内容 | 長文 | ✅ |
| 9 | 個人情報の取り扱いへの同意 | チェック | ✅ |

回答があるたびに `NOTIFY_EMAIL` 宛に通知メールが届きます（送信時トリガー）。

### 2. サイトに反映する

`src/pages/contact.astro` の先頭にある定数を書き換える:

```ts
const FORM_VIEW_URL = 'https://docs.google.com/forms/d/e/XXXXXXXXXXXX/viewform';
```

- 空文字のままだと「準備中」表示
- 公開URLを入れると、**埋め込みフォーム** と **QRコード** が自動で本番表示に切り替わる
- QR はビルド時に `uqr` で生成（インラインSVG。外部リクエスト・第三者サービスなし）

デプロイすれば完了です。

### 3. 通知先メールだけ後で変える場合

Apps Script の `NOTIFY_EMAIL` を本番アドレスに書き換えて保存するだけ（フォームの作り直し不要）。
