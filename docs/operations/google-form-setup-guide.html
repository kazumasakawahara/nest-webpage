1Form を新規作成（5分）

  1. 事務局共用アカウントで Google にログイン
  2. https://forms.google.com を開く → 「空白のフォーム」（＋）
  3. タイトル：NPO法人 nest 入会申込フォーム
  4. 説明欄に以下を貼り付け：
▼ フォーム説明欄にコピペNPO法人 nest の活動を支えてくださる会員を募集しています。 下記フォームにご記入のうえ送信してください。 ご不明な点は事務局（093-582-7018／平日 8:00〜20:00）まで。 ※ 紙の申込書をご希望の方は、こちらから印刷できます： https://www.nponest.org/pdfs/forms/membership-application.html

3設定・デザイン（10分）

設定タブ
  • 「回答」セクション → 「回答のコピーを送信」を 「リクエストされた場合」 または 「常時」 に設定（申込者が控えを受け取れる）
  • 「メールアドレスを収集する」→ 「確認済み」 推奨（なりすまし防止）
  • 「ログインの要求」→ OFF（会員以外も申込できるように）
デザイン（パレットアイコン）
  • テーマカラー：カスタム → #2D5A27（nest 緑）
  • 背景色：薄いグリーン or クリーム
  • ヘッダー画像：NEST LOGO_01.png をアップロード（任意）
  • フォント：基本フォント（日本語が読みやすいもの）

4サンクスメッセージ（送信後の表示）（5分）

「設定」→「プレゼンテーション」→「確認メッセージ」を編集：

▼ 確認メッセージにコピペお申し込みありがとうございます。 内容を確認のうえ、事務局よりご連絡いたします。 【会費のお支払いについて】 当面は現金でのお支払いをお願いしております。 事務局へご持参いただくか、行事の際にお渡しください。 （オンライン決済は今後対応予定です） ・正会員：入会金 ¥1,000 ＋ 年会費 ¥2,000 ・賛助会員（個人・団体）：年会費 ¥2,000 ご不明な点は事務局まで：093-582-7018（平日 8:00〜20:00）

5スプレッドシート連携（5分）

  1. 「回答」タブ → スプレッドシートのアイコン（緑）をクリック
  2. 「新しいスプレッドシートを作成」→ 名前：入会申込_回答
  3. 作成されたスプレッドシートを開く
  4. 質問列の右側に、以下の管理用列を手動で追加：
列名用途
受付状況未対応 / 確認中 / 完了（手動更新）
会費確認未納 / 入金確認済 / 不要
nest登録日会員管理画面で登録した日
会員番号登録時に発番
事務局メモ備考
「受付状況」列は Apps Script（STEP 6）が新規申込時に自動で「未対応」をセットします。手動で初期値を入れる必要はありません。

6Apps Script で半自動化（Level 2）（20分）

申込があるたびに「①事務局への通知メール」「②申込者への会費案内メール」「③受付状況の初期化」を自動で行います。

  1. STEP 5 のスプレッドシートを開く
  2. メニュー「拡張機能」→「Apps Script」
  3. 既存のコードを全削除し、以下を貼り付け
  4. OFFICE_EMAIL を事務局のメールアドレスに変更
  5. 保存（💾）
// ===== NPO nest 入会申込フォーム — 自動処理（Level 2） =====
// ▼ 事務局のメールアドレスに変更してください
const OFFICE_EMAIL = 'jimukyoku@nponest.org';

function onFormSubmit(e) {
  const r = e.namedValues;
  const name  = (r['お名前（団体の場合は団体名）'] || [''])[0];
  const email = (r['メールアドレス'] || [''])[0];
  const type  = (r['会員区分'] || [''])[0];

  // --- ① 事務局への通知 ---
  const lines = Object.keys(r).map(k => k + '：' + r[k].join(', '));
  MailApp.sendEmail(
    OFFICE_EMAIL,
    '【入会申込】' + name + ' 様（' + type + '）',
    '新しい入会申込がありました。\n\n' + lines.join('\n') +
    '\n\nスプレッドシートで詳細を確認してください。'
  );

  // --- ② 申込者への会費案内（自動返信） ---
  if (email) {
    const fee = (type.indexOf('正会員') >= 0)
      ? '入会金 ¥1,000 ＋ 年会費 ¥2,000（初年度 ¥3,000、次年度以降 ¥2,000）'
      : '年会費 ¥2,000';
    MailApp.sendEmail(
      email,
      '【nest】入会申込を受け付けました',
      name + ' 様\n\n' +
      'この度は NPO法人 nest への入会をお申し込みいただき、ありがとうございます。\n' +
      '以下の内容で受け付けました。\n\n' +
      '■ 会員区分：' + type + '\n' +
      '■ お支払い金額：' + fee + '\n\n' +
      '【お支払い方法】\n' +
      '当面は現金でのお支払いをお願いしております。\n' +
      '事務局までご持参いただくか、行事の際にお渡しください。\n' +
      '（オンライン決済は今後対応予定です）\n\n' +
      'ご不明な点は事務局まで：093-582-7018（平日 8:00〜20:00）\n\n' +
      'NPO法人 nest'
    );
  }

  // --- ③ 受付状況を「未対応」に初期化 ---
  const sheet = e.range.getSheet();
  const row = e.range.getRow();
  const headers = sheet.getRange(1, 1, 1, sheet.getLastColumn()).getValues()[0];
  let col = headers.indexOf('受付状況') + 1;
  if (col === 0) { col = sheet.getLastColumn() + 1; sheet.getRange(1, col).setValue('受付状況'); }
  sheet.getRange(row, col).setValue('未対応');
}
トリガーの設定（重要）
  1. Apps Script 画面の左メニュー「トリガー」（⏰アイコン）
  2. 右下「トリガーを追加」
  3. 設定：
    • 実行する関数：onFormSubmit
    • イベントのソース：スプレッドシートから
    • イベントの種類：フォーム送信時
  4. 保存 → Google アカウントの承認を求められるので許可
承認画面で「このアプリは確認されていません」と出ますが、これは自作スクリプトのため正常です。「詳細」→「（プロジェクト名）に移動」→「許可」で進めてください。事務局共用アカウントの所有者が実施します。

7テスト → URL を河原さんへ（10分）

テスト送信
  1. Form 右上「プレビュー」（👁）→ 自分で1件テスト申込
  2. 確認：
    • スプレッドシートに行が追加されたか
    • 「受付状況」が自動で「未対応」になったか
    • 事務局アドレスに通知メールが届いたか
    • 申込者アドレス（テストで入れたもの）に会費案内メールが届いたか
  3. テスト行はスプレッドシートから削除しておく
公開 URL の取得
  1. Form 右上「送信」ボタン → リンクアイコン（🔗）
  2. 「URL を短縮」にチェックを入れると短い URL になる
  3. 表示された URL をコピー
