## 1. 背景と目的

ナビゲーション（`src/lib/site.ts` の `navLinks`）には既に「グループホーム」リンクがあり `/group-home/` を指しているが、現状は「ただいま準備中です」のスタブが公開されている。これを、内容は **案内程度（軽量）** に留めつつ、nest のグループホーム運営の理念を伝えるページに置き換える。

**伝えたい核となるメッセージ**：

nest は、利用者を一か所の施設に集めるのではなく、**町内に点在する賃貸マンションの居室を法人で借り上げて** グループホームを運営している。これにより利用者は隔離された特別な環境ではなく、近所付き合いや買い物といった **当たり前の地域生活** の中で暮らせる。これが nest の理念「巣立ち＝地域で生きる」の実践である。

分散した居室で運営するもう一つの理由は、**利用者の中に強いこだわりを持つ方も少なくなく、集団生活が難しいケースがある** こと。分散した住まいだからこそ、一人ひとりの個性とペースを尊重できる。

## 2. プライバシー方針（重要）

- 個別の **住所・拠点名・居室の定員・利用者氏名は一切掲載しない**。
- 拠点数は **数のみ**（「町内に12の住まいを運営」）で言及する。
- STATION / BRANCH / SATELLITE は **概念（住まいの段階）として** 紹介し、`about.astro` にあるような個別拠点名リスト（原町STATION 等）は載せない。

## 4. 本文コピー（草案・確認後に確定）

### 理念ブロック

> nest のグループホームは、ひとつの施設に利用者を集めて運営するものではありません。わたしたちは、町内に点在する賃貸マンションの居室を法人で借り上げ、そのひと部屋ひと部屋を利用者の住まいとしています。
>
> だからこそ、利用者は特別な場所に隔離されることなく、ご近所との何気ない挨拶や、近くのお店での買い物といった、当たり前の地域の暮らしの中で日々を送ることができます。「本人が地域で暮らす」——それが、わたしたち nest の願いです。
>
> また、利用者の中には強いこだわりを持つ方も少なくありません。大人数での集団生活がむずかしい方も、分散した住まいであれば、一人ひとりの個性とペースを尊重しながら、その人らしい暮らしを支えることができます。

### 住まいの段階カード

- STATION：世話人が常駐する拠点です。手厚い支援を受けながら、暮らしの基盤をつくります。
- BRANCH：STATION の近くにある居室です。サポートを受けつつ、少しずつ自分のペースで暮らします。
- SATELLITE：より一人暮らしに近い住まいです。自立した生活を見据えて、その人らしく過ごします。
